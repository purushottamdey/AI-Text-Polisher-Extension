üêõ TROUBLESHOOTING
------------------
Problem: Button doesn't appear when selecting text
Solution: 
- Refresh the page after installing
- Make sure Developer mode is enabled
- Check the extension is enabled in chrome://extensions/

Problem: API Error messages
Solution:
- Verify your API key is correct
- Check you have credits with your AI provider
- Try a different model name

Problem: Text won't replace
Solution:
- Some fields (passwords, etc.) cannot be modified
- Use the "Copy" button instead
- Manually paste the improved text
