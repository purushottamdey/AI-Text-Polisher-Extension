═══════════════════════════════════════════════════
COMMON PROBLEMS & INSTANT FIXES
═══════════════════════════════════════════════════

Problem: "No button appears when I select text"
Fix:
1. Press F12, go to Console tab
2. Select text again
3. Do you see ANY red errors?
4. If yes, tell me the error
5. If no errors, reload extension and try again


Problem: "Extension icon doesn't appear in toolbar"
Fix:
1. Click puzzle piece icon (top right of Chrome)
2. Find "AI Text Polisher"
3. Click the pin icon to pin it to toolbar


Problem: "Can't click extension icon, nothing happens"
Fix:
1. Go to chrome://extensions/
2. Click "Details" on AI Text Polisher
3. Scroll down
4. Make sure "Site access" is set to "On all sites"


Problem: "Button appears but clicking does nothing"
Fix:
1. Go to chrome://extensions/
2. Find "AI Text Polisher"
3. Look for blue link "service worker"
4. Click it
5. Check for errors in that console
6. Tell me what errors you see


Problem: "Settings popup won't open"
Fix:
1. Right-click the extension icon
2. Select "Manage extension"
3. Check for errors on that page
